功能实现说明

递归函数 countDown(seconds, callback)

使用递归和setTimeout模拟异步倒计时，每秒打印剩余秒数，当seconds为0时调用回调函数。

function countDown(seconds, callback) {
  if (seconds >= 0) {
    console.log(`剩余: ${seconds}秒`);
    if (seconds > 0) {
      setTimeout(() => countDown(seconds-1, callback), 1000);
    } else {
      callback();
    }
  }
}

闭包函数 createTimer()

使用闭包创建私有变量startCount，返回包含start和getStartCount方法的对象，保护内部状态不被外部直接修改。

function createTimer() {
  let startCount = 0; // 私有变量
  return {
    start(seconds, callback) {
      startCount++;
      countDown(seconds, callback);
    },
    getStartCount() {
      return startCount;
    }
  };
}
